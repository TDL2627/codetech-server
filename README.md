#TITLE 
      -userAdmin

#INTRODUCTION
      - This project consists of node js and firestore functions to create a sign up,that differentiates admins and normal users,validates input and the use of CRUD methods.It also assigns and object of skills to each user.

#TECHNOLOGIES
      -express js
      -node js
      -firebase 
      -Javascript
      -Postman

#LAUNCH
      -install npm:
                   npm init
                   npm i express --s
                   npm i firebase-admin -s
                   npm i -g nodemon
                   npm i body-parser --s
                   npm i --save express-validator
     -install Postman or Thunder client on vs code to run commands
     -run "nodemon dev" in terminal to check if connected to server
     -create Firebase project and web app

#FEATURES
        -can post users to database
        -can check user input
        -can delete ,edit and add data to the database via the server
        -can assign data to users within the database
